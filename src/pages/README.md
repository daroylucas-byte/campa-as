# Páginas (Views)

Esta carpeta contendrá las pantallas principales de la aplicación:
- Login / Registro
- Selector / Selector de Negocio
- Panel de Clientes (Clientes, Identidad Visual)
- Panel de Campañas (Planificación de 30 días, posts semanales, editor de post)
- Billetera / Wallet (Carga de saldo y transacciones)

Toda la interacción con el backend se realiza a través de `src/services/`.
